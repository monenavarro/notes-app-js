User Stories

```
As a busy person, so that I can remember the things that I need to do, 
I want to see a list of tasks on my toDo list.

As a busy person, so that I can remember something that I need to do,
I want to be able to add a new task on my toDo list.

As a busy person, so that I can know when I have completed all of my tasks,
I want to be able to clear all of the tasks on my toDo list.
```
